**Date:** 2026-07-14

## Reference

## Influence and provenance

The reference was reviewed for its general approach to rendering a dense, moving character field smoothly. It informed the decision to use a bounded GPU glyph-atlas and instanced-rendering path for the decorative Hero text field.

No source code, shader, assets, text, branding, layout, or visual composition was copied. Scra Atlas keeps its existing circular-arc geometry, technical-copy tracks, palette, typography, `·` gap treatment, and pointer interaction semantics.
